# Swarna Avvaru Portfolio

Personal portfolio site for Swarna Avvaru, a software engineer building enterprise-sca  le web applications and reliable cloud-native solutions.

The site highlights technical strengths, selected projects, certifications, and contact links for professional networking and job-search visibility.

## Live Site

[https://swarnakishore.github.io](https://swarnakishore.github.io)

## Sections

- Hero summary and professional positioning
- Technical expertise across frontend, backend, cloud, testing, and AI-assisted development
- Selected projects, including AI Incident Copilot
- Certifications and professional links
- Contact section with email, LinkedIn, and GitHub

## Tech Stack

- HTML
- CSS
- JavaScript
- Bootstrap 4
- Font Awesome
- GitHub Pages

## Run Locally

From the repository root:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

The site is static, so no build step is required.

## Project Structure

```text
.
├── index.html
├── css/
│   └── portfolio.css
├── js/
│   └── scripts.js
└── assets/
    └── img/
```

## Updating Content

- Update page content in `index.html`
- Update layout and visual styling in `css/portfolio.css`
- Keep JavaScript minimal in `js/scripts.js`
- Add new images under `assets/img/`
